# Python Data Quality Checker

A simple data quality project that checks for missing values, duplicates, invalid dates, and negative values.

## What this project does
- Reads employee records from a CSV file
- Checks common data quality issues
- Creates a cleaned output file
- Writes a quality report to the console

## Tech stack
- Python
- pandas

## Resume bullets
- Developed a Python-based data quality checker to identify missing values, duplicates, invalid dates, and inconsistent records
- Automated dataset validation and generated cleaned output files for downstream analytics
- Improved data reliability by applying reproducible validation rules and transformation logic

## How to run
```bash
pip install -r requirements.txt
python quality_check.py
```
