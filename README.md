# SQL Sales Analysis

A SQL portfolio project that demonstrates schema creation, data loading, and business analysis queries.

## What this project includes
- Table creation script
- Sample data inserts
- Analytical SQL queries using:
  - JOINs
  - GROUP BY
  - HAVING
  - Window functions

## Tech stack
- SQL
- PostgreSQL or MySQL-compatible syntax with minor edits

## Resume bullets
- Designed a relational schema for sales and customer data and populated it with sample transactional records
- Wrote analytical SQL queries using joins, aggregations, CTEs, and window functions to identify revenue trends and top customers
- Built query-based insights for monthly sales reporting and product performance analysis

## Suggested GitHub repo structure
- `schema.sql`
- `sample_data.sql`
